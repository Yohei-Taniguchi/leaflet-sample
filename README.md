#地図アプリサンプル
## 内容
- 和歌山県が提供する避難所データを利用して、Leaflet.jsを使って国土地理院が提供する地図上に避難場所をマッピング
- 避難所から半径500mを円で表示

## 利用技術・サービス
- [Leaflet](http://leafletjs.com/)
- [国土地理院地図](http://maps.gsi.go.jp/development/ichiran.html)
- [和歌山県避難所データ](https://github.com/wakayama-pref-org)
